**核心概述**: 提出概率型用户行为模型：先建全局模型，再对已有用户个性化。

**中文详细解读**: 从题目与摘要可见，论文研究的是用户行为建模问题，核心目标是用概率模型刻画用户行为。其基本思路是：先为整个用户群体建立一个全局行为模型，再在此基础上对已有用户进行个性化，即为每个用户分配相应的个体化表示或参数。论文的主要贡献应在于把“群体层面的统一建模”与“用户层面的个性化调整”结合起来，以同时利用总体规律和个体差异。就目前提供的信息而言，能够明确的方法只有“全局模型+个性化”这一框架；摘要片段未给出更具体的模型形式、训练过程、数据集设置及实验结果，因此实验效果与定量结论无法据此展开。

**相关应用**: 1. 面向 user behavior modeling 的用户行为预测与刻画；2. 基于全局模型与个体差异的已有用户个性化建模；3. 群体行为规律分析与用户级行为表示。

**阅读建议**: 值得阅读，尤其适合关注 user behavior modeling、概率模型、个性化建模的读者。若你的研究重点是“如何同时利用群体共性与个体差异”，这篇论文与关键词结合紧密。由于当前仅有题目与摘要片段，建议阅读全文时重点关注：全局模型如何构建、用户如何被个性化分配、实验如何验证有效性。可能的改进方向需结合全文方法与实验后再判断。
